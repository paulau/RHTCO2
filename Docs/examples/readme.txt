To get visualisation of sensors data:
    1 copy the *.std files from raspberry to this folder
    2 copy content of srcvisu/ to the folder of python libraries e.g. c:/Python34/Lib/ or to this folder
    3 copy settingsRHTCO2_009.py to this folder
    4 run to concatenate:
    monat1.py settingsRHTCO2_009.py 
    5 set visualisation interval in settingsRHTCO2_009.py file d1 d2 parameters
    and start visualisation
    visualiserotrocall.py aaa01_2018_09_12_00_00.std
    
    The result is saven into the same folder. 
    File With jpg extension aaa01_2018_09_12_00_00.jpg
